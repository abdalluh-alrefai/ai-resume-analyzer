# AI Resume Analyzer Pro

AI-powered resume analysis platform built with **Python** and **Streamlit**.

This application helps users analyze resumes, calculate ATS compatibility, identify strengths and weaknesses, generate improved resume summaries, build resumes, create templates, and generate cover letters.

The project demonstrates practical AI-assisted resume optimization and is designed as a **portfolio-ready micro SaaS project**.

# Features

### Resume Analysis

* Resume score
* ATS compatibility score
* Strengths detection
* Weaknesses detection
* Skills extraction
* Resume summary generation
* AI resume rewrite
* ATS improvement suggestions

### Job Matching

* Detect matched skills

# Installation

Clone the repository

```
git clone https://github.com/abdalluh-alrefai/ai-resume-analyzer.git
```

Go into the project folder

```
cd ai-resume-analyzer
```

Install dependencies

```
pip install -r requirements.txt
```

Run the application

```
streamlit run app.py
```
